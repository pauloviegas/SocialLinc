Boolean alterar( Array $dados, [Array $parametroAlteracao = NULL])

Altera as informações da tabela setada na classe filha, atraves do id, caso não seja informado o segundo parâmetro, ou atraves do segundo parâmetro, caso este seja populado.
OBS: No Array $dados deve estar contido o id a ser alterado, caso se deseje fazer a alteração pelo mesmo.

method deletar [line 115]

Boolean deletar( [Int $id = NULL], [Array $parametros = NULL])

Exclui logicamente (alterando para 0 o campo excluido do registro), ou fisicamente, um ou mais registros da tabela setada na classe filha, a partir de id, caso ele exista, ou a partir do segundo parâmetro, caso o id seja NULL e segundo parâmetro seja populado.
OBS 1: Se houver mais de um registro que correspondam aos valores dos parâmetros de busca passados, todos serão excluidos.

OBS 2: Se os dois parâmetros forem setados os dois serão parâmetros para busca.




Tags:

return:  Valor lógico, onde 1 corresponde a sucesso e 0 a erro.
author:  Paulo Viegas <pauloviegas93@gmail.com>
access:  public


Parameters:

Int   $id   corresponde ao id linha a ser excluida;
Array   $parametros   A chave corresponde a coluna da tabela no banco e o valor da chave corresponde ao valor do filtro a ser usado na exclusao.

Array recuperaPorParametro( [Int $id = NULL], [Array $parametrosBusca = NULL])
